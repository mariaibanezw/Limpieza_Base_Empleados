# Descripción del proyecto

En este proyecto se realizó la limpieza y normalización de un conjunto de datos de los empleados de una empresa, almacenado en un archivo CSV. Los datos contenían inconsistencias en nombres de columnas, valores faltantes y posibles errores tipográficos, por lo que se implementaron procesos en SQL Server para mejorar su calidad y estructura.

# Objetivo general

Optimizar la calidad de los datos de empleados de una empresa mediante la limpieza, normalización y estandarización de la información en SQL Server.

# Herramientas utilizadas

SQL Server: Limpieza, transformación y modelado de los datos.

# Resultados obtenidos

- Datos estandarizados y listos para análisis.
- Eliminación de inconsistencias y mejora en la calidad del dataset.
- Base de datos más eficiente y organizada.
